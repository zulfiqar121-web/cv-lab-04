# Computer Vision Lab 4

This project is part of a Computer Vision laboratory assignment. The goal of this lab is to understand how image data can be processed and analyzed using machine learning and deep learning techniques.

The project focuses on building a basic image classification system using Python and common computer vision libraries. It demonstrates the complete workflow of a computer vision task, starting from data preparation to model training and evaluation.

## Project Overview

In this lab, an image dataset is used to train a model that can recognize and classify different types of images. The project covers the following steps:

1. Loading and preparing image data
2. Preprocessing images for model training
3. Building a classification model
4. Training the model on the dataset
5. Evaluating the model performance

This lab helps in understanding how computers can interpret visual information and make predictions based on patterns found in images.

## Technologies Used

The following tools and libraries are used in this project:

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Jupyter Notebook

These technologies are commonly used in computer vision and machine learning projects.

## Project Structure

The repository contains the following main components:

Lab_4.ipynb  
The main Jupyter Notebook that contains the implementation of the computer vision model, including data preprocessing, training, and evaluation.

Dataset  
The folder containing the images used for training and testing the model.

README.md  
Documentation explaining the purpose and structure of the project.
